# AI-Powered Failure Analysis

This project demonstrates how **PowerShell scripting + GitHub Copilot CLI** can be used to analyze CI/CD pipeline logs, while **Python automation** collects logs from multiple folders into one central location.

## 🚀 Features
- PowerShell script integrates with Copilot CLI for AI-powered log analysis
- Python script gathers logs from multiple projects
- Lightweight, recruiter-friendly demo of automation + AI
